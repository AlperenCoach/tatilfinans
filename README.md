# TatilFinans

TatilFinans - Turizm ve otel rezervasyon web sitesi.


## Kurulum

1. Bağımlılıkları yükleyin:
```bash
npm install
```

2. Geliştirme sunucusunu başlatın:
```bash
npm start
```

3. Tarayıcınızda `http://localhost:3000` adresine gidin.

## Teknolojiler

- React 18
- React Icons
- CSS3
