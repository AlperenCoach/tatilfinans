# TatilFinans

TatilFinans - Turizm ve otel rezervasyon web sitesi.

## Özellikler

- Modern ve responsive tasarım
- 5 resimlik otomatik slider
- Sidebar menü
- Detaylı footer
- Navbar navigasyon fonksiyonları

## Kurulum

1. Bağımlılıkları yükleyin:
```bash
npm install
```

2. Geliştirme sunucusunu başlatın:
```bash
npm start
```

3. Tarayıcınızda `http://localhost:3000` adresine gidin.

## Renk Teması

- Ana Renk: #1a4088
- Vurgu Rengi: #ffd700 (Sarı)
- Arka Plan: Beyaz
- Metin: Siyah

## Teknolojiler

- React 18
- React Icons
- CSS3
